Role Name
=========

Installs and configures Sensu monitoring.  https://sensuapp.org/

Requirements
------------

Install all requirements using Ansible Galaxy.  
````
sudo ansible-galaxy install -r requirements.yml -f
````

Vagrant
-------

Spin up environment using Vagrant.  
````
vagrant up
````
